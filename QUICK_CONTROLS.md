## Quick Tips

**New to Newtonian Physics?**
- Start with Assist on **HIGH** (green)
- Remember: You keep moving after releasing keys!
- To stop: thrust in the **opposite direction**
- To turn: point nose + thrust to change velocity

**Getting the Hang of It?**
- Drop to **MEDIUM** assist (blue)
- Practice drift maneuvers around asteroids
- Learn to use lateral thrusters (A/D)

**Advanced Pilots:**
- Set assist to **OFF** (red) for pure sim
- Master angular momentum control
- Execute precision high-speed maneuvers

**Controls Feel Wrong?**
- Mouse sensitivity can be adjusted in Settings
- Y-axis invert available
- Try different assist levels

**Lost Control?**
- Press **T** until assist is **HIGH** (green)
- Release all controls, let assist stabilize
- Press **ESC** if you need a break
